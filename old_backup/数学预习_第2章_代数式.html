2.1 用字母表示数

📖 基础概念

用字母表示数，可以把数量和数量之间的关系一般化，这是从算术到代数的重要一步。

书写规范（考试经常扣分！）：

  • 数字与字母相乘，数字写前面，2×a 写作 2a
  • 字母与字母相乘，直接写在一起，a×b 写作 ab
  • 数字与字母相除，写成分数形式，a÷2 写作 a/2
  • "1" 作为系数时省略：1a 写作 a
  • 带分数要先化成假分数：2⅓a = 7/3 a

📝 基础例题

用代数式表示：

(1) a 与 b 的和的 2 倍    (2) a 的 3 倍与 b 的 1/2 的差

(1) "和的 2 倍"：先求和，再乘 2 → 2(a+b)（注意加括号！）

(2) "a 的 3 倍" = 3a，"b 的 1/2" = 1/2 b，相减 → 3a - 1/2 b

⚠️ 易错提醒

  • "a 与 b 的和的 2 倍" 是 2(a+b)，不是 2a+b！
  • 除法结果必须写成分数形式，不能保留"÷"号。
  • 带分数必须先化成假分数再运算。

📝 即时练习（5道题）

练习1：用代数式表示：a 与 b 的和的相反数。

答：-(a + b)

练习2：长方形的宽是 a 厘米，长比宽多 3 厘米，用代数式表示周长和面积。

答：长 = a + 3，周长 = 2(a + a + 3) = 4a + 6 厘米，面积 = a(a + 3) 平方厘米

练习3：下列说法正确的是（ ）

A. a×2 写作 a2
B. a÷2 写作 a/2
C. 1×a 写作 1a
D. 2×3 写作 23

练习4：一辆汽车以每小时 v 千米的速度行驶，t 小时后行驶多少千米？

答：vt 千米

练习5：用代数式表示：x 的 2 倍与 y 的 3 倍的和。

答：2x + 3y

2.2 代数式的概念

📖 基础概念

代数式：用运算符号（加、减、乘、除、乘方）把数或表示数的字母连接而成的式子。

注意：单独的一个数或一个字母也是代数式（如 5, a, x）。

📝 基础例题

判断下列哪些是代数式：

3x + 5, 2a = 6, S = πr², 0, -x

代数式：3x + 5, 0, -x

不是代数式：2a = 6（等式）, S = πr²（等式）

关键：代数式不含等号或不等号。

📝 即时练习（5道题）

练习1：下列各式中，哪些是代数式？（多选）

A. 2x + 3    B. 5    C. a    D. x > 3    E. x + y = 5

答：A, B, C（D 是不等式，E 是等式，都不是代数式）

练习2：代数式 (a + b) / 2 表示的是（ ）

A. a 与 b 的和
B. a 与 b 的差
C. a 与 b 的平均值
D. a 与 b 的积

练习3：用代数式表示：棱长为 a 的正方体的体积。

答：a³

练习4：下列说法正确的是（ ）

A. 代数式可以含等号
B. 单独的一个数不是代数式
C. 单独的一个字母是代数式
D. 代数式 x > 2 是正确的写法

练习5：用代数式表示：半径为 r 的圆的周长和面积。

答：周长 = 2πr，面积 = πr²

2.3 代数式的值

📖 基础概念

代数式的值：用数值代替代数式里的字母，计算后得到的结果。

步骤：① 写出字母的值 → ② 代入代数式 → ③ 按运算顺序计算

注意：代入负数或分数时，一定要加上括号！

📝 基础例题

当 a = 3, b = -2 时，求 a² - 2ab + b² 的值。

解：

原式 = 3² - 2×3×(-2) + (-2)²

= 9 - (-12) + 4

= 9 + 12 + 4 = 25

⚠️ 易错提醒

  • 代入负数时千万别忘了加括号！
  • 错误：a = -2 代入 a² 时写成 -2² = -4（错！）
  • 正确：(-2)² = 4

📝 即时练习（5道题）

练习1：当 x = 2, y = -3 时，求 2x² - 3y + 1 的值。

解：原式 = 2×(2)² - 3×(-3) + 1 = 8 + 9 + 1 = 18

练习2：当 a = -1, b = 2 时，求 (a + b)(a - b) 的值。

解：原式 = (-1 + 2)(-1 - 2) = 1 × (-3) = -3

练习3：当 x = 1/2 时，求 4x - 3 的值。

解：原式 = 4×(1/2) - 3 = 2 - 3 = -1

练习4：当 a = -2, b = -3 时，求 a² + 2ab + b² 的值。

解：原式 = (-2)² + 2×(-2)×(-3) + (-3)² = 4 + 12 + 9 = 25

练习5：当 x = -1 时，求 -x² + 2x - 1 的值。

解：原式 = -(-1)² + 2×(-1) - 1 = -1 + (-2) - 1 = -4

注意：-x² = -(x²)，不是 (-x)²！

2.4 一次式的概念

📖 基础概念

单项式：只含有乘法（和乘方）运算的代数式叫做单项式。

多项式：几个单项式的和叫做多项式。

一次式：次数为 1 的整式（单项式或多项式）。

⚠️ 易错提醒

  • x + y 不是一次式（含两个字母）
  • x² 不是一次式（指数为 2）
  • 3x + 2y - 5 不是一次式（含两个字母）

📝 即时练习（5道题）

练习1：下列各式中，哪些是单项式？（多选）

A. 3x    B. -2xy²    C. 3x + 2y    D. 5    E. a/b

答：A, B, D（C 是多项式，E 含有除法，不是单项式）

练习2：单项式 -2xy² 的系数和次数分别是（ ）

A. -2, 2
B. -2, 3
C. 2, 2
D. 2, 3

练习3：下列式子中，哪些是一次式？（多选）

A. 3x    B. -2x + 5    C. x + y    D. 2x + 3y - 1    E. x²

答：A, B（C 和 D 含两个字母，不是一次式；E 次数为 2，不是一次式）

练习4：多项式 3x² - 2x + 5 的次数是______。

答：2（次数最高的项是 3x²，次数为 2）

练习5：单项式 1/2 ab²c 的次数和系数分别是______和______。

答：次数 = 4（1+2+1），系数 = 1/2

2.5 一次式的同类项

📖 基础概念

同类项：所含字母相同，并且相同字母的指数也相同的项。

关键：① 字母相同 ② 相同字母的指数相同（与系数无关，与字母顺序无关）

📝 基础例题

下列各组式子中，哪些是同类项？

(1) 3x²y 与 -2x²y    (2) 2ab 与 3ba    (3) 5x² 与 5x³

(1) 是同类项（字母相同，x 的指数都是 2，y 的指数都是 1）

(2) 是同类项（字母相同，与顺序无关）

(3) 不是同类项（x 的指数不同）

⚠️ 易错提醒

  • 3x²y 和 -2xy² 不是同类项！虽然都有 x 和 y，但指数不同。
  • 常数项都是同类项（比如 3 和 -5 是同类项）。

📝 即时练习（5道题）

练习1：下列各组式子中，是同类项的有（ ）（多选）

A. 3x 与 -2x    B. 2ab 与 2a    C. 5 与 -3    D. x²y 与 xy²

答：A, C（B 字母不同，D 指数不同）

练习2：合并同类项：3x² - 2x + 5 - x² + 4x - 3

解：原式 = (3x² - x²) + (-2x + 4x) + (5 - 3) = 2x² + 2x + 2

练习3：下列说法正确的是（ ）

A. 字母相同的项是同类项
B. 常数都是同类项
C. x²y 与 xy² 是同类项
D. 系数相同的项是同类项

练习4：如果 3xᵐy² 与 -2x³yⁿ 是同类项，那么 m = ______，n = ______。

答：m = 3，n = 2（同类项要求相同字母的指数相同）

练习5：合并同类项：-a + 2b - 3a + 5b = ______。

答：-4a + 7b

2.6 一次式的加减

📖 基础概念

去括号法则：

  • 括号前是"+"号：去掉括号和"+"号，括号内各项不变号
  • 括号前是"-"号：去掉括号和"-"号，括号内各项都变号

整式加减的步骤：① 如果有括号，先去括号 → ② 合并同类项

📝 基础例题

计算：(3a² - 2ab + 5) - (a² + 4ab - 3)

先去括号（第二个括号前是"-"，里面各项变号）：

= 3a² - 2ab + 5 - a² - 4ab + 3

合并同类项：

= (3a²-a²) + (-2ab-4ab) + (5+3)

= 2a² - 6ab + 8

💡 解题技巧

  • 用不同符号标记不同类型的同类项
  • 移项时要带符号一起移动
  • 结果是一次式时，按字母顺序排列

📝 即时练习（5道题）

练习1：计算：(2x + 3) - (x - 5) = ______。

答：x + 8（计算：2x + 3 - x + 5 = x + 8）

练习2：计算：3(a + 2b) - 2(2a - b) = ______。

答：-a + 8b（计算：3a + 6b - 4a + 2b = -a + 8b）

练习3：去括号：-2(3x - 5y + 1) = ______。

答：-6x + 10y - 2

练习4：计算：(5x² - 3x + 2) + (2x² + 4x - 1) = ______。

答：7x² + x + 1

练习5：计算：(4a - 3b) - (2a + 5b) = ______。

答：2a - 8b

2.7 数与一次式相乘

📖 基础概念

分配律：k(a + b) = ka + kb，这是数与一次式相乘的核心法则。

步骤：① 用数字因数乘以括号内的每一项 → ② 注意符号！

📝 基础例题

计算：-3(2x - 5y + 1)

用分配律：

= (-3)×2x + (-3)×(-5y) + (-3)×1

= -6x + 15y - 3

⚠️ 易错提醒

  • 分配律时，数字因数要乘以每一项，不能漏项！
  • 符号最容易错：-2(3-5) = -6+10 = 4，不是 -6-10 = -16！

📝 即时练习（5道题）

练习1：计算：2(3x - 4y) = ______。

答：6x - 8y

练习2：计算：-4(x - 2y + 3) = ______。

答：-4x + 8y - 12

练习3：计算：1/2(4x - 6y) = ______。

答：2x - 3y

练习4：先去括号，再合并同类项：3(x + 2) - 2(x - 1) = ______。

答：x + 8（计算：3x + 6 - 2x + 2 = x + 8）

练习5：下列说法正确的是（ ）

A. -2(3x - 1) = -6x - 2
B. -2(3x - 1) = -6x + 2
C. -2(3x - 1) = -5x
D. -2(3x - 1) = -6x - 1
