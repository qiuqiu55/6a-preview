第1章 有理数

📚 选择小节

1.1 正数与负数

📖 基础概念

正数：大于 0 的数（如 +3, +5.2），"+" 通常可省略。

负数：小于 0 的数（如 -3, -5.2），"-" 不能省略。

0 的特殊性：0 既不是正数也不是负数，是正负数的分界点。

📝 基础例题

指出下列哪些是正数，哪些是负数：

+8, -3, 0, +2.5, -4/5, 100, -0.01

正数：+8, +2.5, 100

负数：-3, -4/5, -0.01

既不是正数也不是负数：0

💡 题型变化

变化1：a 一定是正数吗？答：不一定。a 可以表示任意数。

变化2：如果收入 200 元记作 +200 元，那么支出 50 元记作______元。答：-50

变化3：如果 m > 0，那么 -m 是______数。答：负数

⚠️ 易错提醒

  • 0 不是没有温度/没有高度！0°C 是一个确定的温度。
  • "-a" 不一定是负数！如果 a 本身是负数，那么 -a 就是正数。
  • 正数前面的 "+" 可以省略，但负数前面的 "-" 绝对不能省略。

📝 即时练习（5道题）

练习1：如果水位升高 3 米记作 +3 米，那么水位下降 2 米记作______米，水位不升不降记作______米。

答：-2 米；0 米

练习2：下列说法正确的是（ ）

A. 0 是最小的正整数
B. 负数一定小于正数
C. 负号可以省略不写
D. 0°C 表示没有温度

练习3：如果 a 是负数，那么 -a 是______数（填"正"或"负"）。

答：正（因为 a < 0，所以 -a > 0）

练习4：下列各数中，哪些是正数？哪些是负数？

-5, +12, 0, -3.2, +1/2, -100

正数：+12, +1/2

负数：-5, -3.2, -100

非正非负：0

练习5：如果向东走 50 米记作 +50 米，那么向西走 30 米记作______米，原地不动记作______米。

答：-30 米；0 米

1.2 数轴

📖 基础概念

数轴的三要素：原点、正方向、单位长度。

数轴是一条直线，通常规定向右为正方向，选取某一长度为单位长度，取一点为原点（表示 0）。

📝 基础例题

在数轴上表示下列各数：+3, -2, 0, +1.5

1. 画一条数轴，标出原点 0

2. 向右为正方向，取单位长度为 1

3. +3 在原点右侧 3 个单位

4. -2 在原点左侧 2 个单位

5. +1.5 在原点右侧 1.5 个单位

💡 题型变化

变化1：数轴上表示 -4 的点到原点的距离是______。答：4

变化2：数轴上表示 +2 的点与表示 -3 的点之间的距离是______。答：5

变化3（动点问题）：一个点从 +1 开始，先向右移动 3 个单位，再向左移动 5 个单位，最后停在______。答：-1

⚠️ 易错提醒

  • 单位长度要一致！数轴上 0 到 1 的距离和 1 到 2 的距离必须相等。
  • 正方向不一定向右！如果题目说明"向左为正方向"，就要按题目来。
  • 表示小数时，先想清楚它在哪两个整数部分之间。

📝 即时练习（5道题）

练习1：在数轴上，到原点距离等于 4 的点所表示的数是______。

答：+4 或 -4（有两个点，分别在原点左右）

练习2：数轴上的点 A 表示 -2，将点 A 向右移动 6 个单位后到达点 B，点 B 表示的数是______。

答：+4（计算：-2 + 6 = 4）

练习3：在数轴上，原点左边的点表示的是（ ）

A. 正数
B. 负数
C. 0
D. 非负数

练习4：数轴上点 M 表示 -5，点 N 表示 +3，这两点之间的距离是______个单位。

答：8（计算：3 - (-5) = 8）

练习5：一个点从数轴上表示 -3 的点开始，向右移动 7 个单位，再向左移动 2 个单位，最后停在______。

答：+2（计算：-3 + 7 - 2 = 2）

1.3 相反数

📖 基础概念

相反数：只有符号不同的两个数，叫做互为相反数。

求一个数的相反数的方法：在这个数的前面加上"-"号，然后化简。

📝 基础例题

写出下列各数的相反数：8, -2/3, 0, -(-5), +(-2)

8 的相反数：-8

-2/3 的相反数：-(-2/3) = 2/3

0 的相反数：0

-(-5) = 5，所以相反数是 -5

+(-2) = -2，所以相反数是 +2

⚠️ 易错提醒

  • "-a" 不一定是负数！如果 a 是负数，-a 就是正数。
  • 多重符号化简时，只数"-"的个数。"+"可以直接忽略。
  • 0 的相反数是 0，这是唯一一个特殊的。

📝 即时练习（5道题）

练习1：化简：-[-(+3)] = ______

答：3（有 2 个"-"号，偶数个，结果为正）

练习2：下列说法正确的是（ ）

A. 符号不同的两个数叫做互为相反数
B. 互为相反数的两个数一定一正一负
C. 如果 a + b = 0，那么 a 和 b 互为相反数
D. 0 没有相反数

练习3：如果 x = -7，那么 -x = ______。

答：7（因为 -(-7) = 7）

练习4：化简：-[-(-a)] = ______（用含 a 的式子表示）

答：-a（有 3 个"-"号，奇数个，结果为 -a）

练习5：如果点 A 表示的数是 a，点 A 关于原点对称的点是 B，那么点 B 表示的数是______。

答：-a（关于原点对称 = 互为相反数）

1.4 绝对值

📖 基础概念

绝对值：一个数在数轴上所对应点到原点的距离，叫做这个数的绝对值。记作 |a|。

代数定义：

  • 当 a > 0 时，|a| = a
  • 当 a = 0 时，|a| = 0
  • 当 a < 0 时，|a| = -a

核心要点：|a| ≥ 0，即绝对值永远是非负数！

📝 基础例题

求下列各数的绝对值：-5, +3, 0, -7/2, +0.6

|-5| = 5

|+3| = 3

|0| = 0

|-7/2| = 7/2

|+0.6| = 0.6

💡 题型变化

变化1（双值性）：已知 |x| = 5，求 x。答：x = 5 或 x = -5

变化2（非负性）：如果 |a-2| + |b+3| = 0，求 a 和 b。答：a=2, b=-3

⚠️ 易错提醒

  • |a| = a 不一定成立！只有当 a ≥ 0 时才成立。
  • |-a| = |a| 恒成立。
  • 已知 |x| = a 求 x 时，不要漏解！x 通常有两个值。

📝 即时练习（5道题）

练习1：如果 |x| = |-7|，那么 x = ______。

答：x = 7 或 x = -7

练习2：已知 |m| = |n|，下列结论正确的是（ ）

A. m = n
B. m = -n
C. m = ±n
D. 以上都不对

练习3：绝对值等于 3 的数有______个，它们是______。

答：2 个；+3 和 -3

练习4：如果 |a| = 4，|b| = 2，且 a < b，那么 a = ______，b = ______。

答：a = -4，b = +2 或 -2（因为 a < b，所以 a 必须是 -4）

练习5：如果 |x-1| = 3，求 x 的值。

答：x-1 = 3 或 x-1 = -3，所以 x = 4 或 x = -2

1.5 有理数大小的比较

📖 基础概念

法则1（数轴法）：在数轴上，右边的点所表示的数总比左边的点所表示的数大。

法则2（符号法）：

  • 正数 > 0 > 负数
  • 两个正数比较：绝对值大的数更大
  • 两个负数比较：绝对值大的反而小！

⚠️ 易错提醒

  • 两个负数比较是考试高频出错点，一定要记住"绝对值大的反而小"。
  • 做比较题时，可以先画数轴，把数标上去，直接看左右关系。
  • 多个数排序时，建议先把所有数分成"负数、零、正数"三类，再分别排序。

📝 即时练习（5道题）

练习1：将 -5, +3, -1, 0, -2.5 从大到小排列：
______ > ______ > ______ > ______ > ______

答：+3 > 0 > -1 > -2.5 > -5

练习2：如果 a = -5/6, b = -6/7，那么 a ______ b（填">"或"<"）。

答：a > b（|a| = 5/6 ≈ 0.833，|b| = 6/7 ≈ 0.857，|a| < |b|，所以 a > b）

练习3：下列各数中，最小的数是（ ）

A. -3
B. -1
C. 0
D. +2

练习4：将 -1/2, -1/3, -1/4 从小到大排列：
______ < ______ < ______

答：-1/2 < -1/3 < -1/4（负数比较，绝对值大的反而小）

练习5：如果 a < 0, b > 0，那么 a ______ b（填">"或"<"）。

答：a < b（负数 < 正数）

1.6 有理数的加法

📖 基础概念

有理数加法法则：

  • 同号两数相加：取相同的符号，并把绝对值相加
  • 异号两数相加：取绝对值较大的数的符号，并用较大的绝对值减去较小的绝对值
  • 一个数同 0 相加：仍得这个数

📝 基础例题

计算：(-7) + (-3), (-7) + (+3), (+7) + (-3), (-7) + (+7)

(-7) + (-3) = -10（同号相加）

(-7) + (+3) = -4（异号相加，|-7| > |+3|）

(+7) + (-3) = +4（异号相加，|+7| > |-3|）

(-7) + (+7) = 0（互为相反数）

⚠️ 易错提醒

  • 先定符号，再算数值！
  • 异号两数相加时，是"大绝对值减小绝对值"，不是"大减小"！
  • 简便运算时，优先把互为相反数或同分母/同类小数结合在一起。

📝 即时练习（5道题）

练习1：计算：(+15) + (-20) + (+12) + (-5) = ______

答：+2（计算：(+15+12) + (-20-5) = +27-25 = +2）

练习2：如果 x 的相反数是 -3，y 的绝对值是 4，且 x + y < 0，求 x + y 的值。

答：x = 3，y = ±4。如果 y = +4，x+y = 7 > 0（不符合）；如果 y = -4，x+y = -1 < 0（符合）。所以 x+y = -1。

练习3：计算：(-12.7) + (+8.9) + (-7.3) + (+1.1) = ______

答：-10（用交换律：(-12.7-7.3) + (8.9+1.1) = -20+10 = -10）

练习4：如果 a > 0, b < 0, |a| < |b|，那么 a + b ______ 0（填">"或"<"）。

答：a + b < 0（异号相加，绝对值大的符号占主导）

练习5：某天早晨气温是 -3°C，中午上升了 8°C，傍晚又下降了 5°C，傍晚的气温是______°C。

答：0°C（计算：-3 + 8 - 5 = 0）

1.7 有理数的减法

📖 基础概念

有理数减法法则：减去一个数，等于加上这个数的相反数。

a - b = a + (-b)

⚠️ 易错提醒

  • 减法变加法时，减号和它后面的数一起变号，不是只变减号！
  • "减负等于加正"：a - (-b) = a + b，这是最常考的变形。
  • 连续减两个数等于减去这两个数的和。

📝 即时练习（5道题）

练习1：计算：(-3) - (+7) - (-5) + (-4) = ______

答：-9（统一成加法：(-3)+(-7)+(+5)+(-4) = -9）

练习2：已知 a = -2, b = -5，求 a - b 的值。

答：a-b = -2-(-5) = -2+5 = 3

练习3：某地某天最高气温是 +8°C，最低气温是 -3°C，这一天温差是______°C。

答：11°C（温差 = 最高温 - 最低温 = 8-(-3) = 11）

练习4：如果 a = -3, b = +5，求 a - b 和 b - a。

答：a-b = -3-(+5) = -8；b-a = 5-(-3) = 8

注：a-b = -(b-a)

练习5：计算：0 - (-5) + (-3) - (+2) = ______

答：0（计算：0+(+5)+(-3)+(-2) = 0）

1.8 有理数的乘法

📖 基础概念

有理数乘法法则：

  • 两数相乘，同号得正，异号得负，并把绝对值相乘
  • 任何数与 0 相乘，积仍为 0

多个有理数相乘：当负因数的个数是偶数时，积为正；当负因数的个数是奇数时，积为负。

📝 基础例题

计算：(-3) × (+4), (-3) × (-4), (+3) × 0

(-3) × (+4) = -12（异号得负）

(-3) × (-4) = +12（同号得正）

(+3) × 0 = 0（有因数 0）

⚠️ 易错提醒

  • 多个因数相乘时，先数负因数的个数定符号，再算数值。
  • 分配律：a(b+c) = ab + ac，注意符号！
  • 0 没有倒数！

📝 即时练习（5道题）

练习1：计算：(-5) × (-3) × (-2) × (+4) = ______

答：-120（负因数有 3 个，奇数，积为负）

练习2：用简便方法计算：99又17/18 × (-9)

答：-899又1/2（把 99又17/18 写成 100 - 1/18，用分配律）

练习3：计算：(-2) × (+3) × (-4) × (-1) = ______

答：-24（负因数有 3 个，奇数，积为负）

练习4：求 -3/5 的倒数。

答：-5/3（倒数 = 1 ÷ (-3/5) = -5/3）

练习5：用简便方法计算：-24 × (1/3 - 3/4 + 5/6)

答：-10（用分配律：-24×1/3 + (-24)×(-3/4) + (-24)×5/6 = -8+18+(-20) = -10）

1.9 有理数的除法

📖 基础概念

有理数除法法则：除以一个数（不等于 0），等于乘以这个数的倒数。

a ÷ b = a × (1/b) (b ≠ 0)

⚠️ 易错提醒

  • 除以一个数 = 乘以它的倒数，注意倒数的符号和原数的符号相同！
  • 0 不能做除数（即分母不能为 0）。
  • 乘除混合运算时，统一化成乘法再定符号。

📝 即时练习（5道题）

练习1：计算：-5/6 ÷ (+2/3) = ______

答：-5/4（计算：-5/6 × 3/2 = -15/12 = -5/4）

练习2：如果 a ÷ b = -1，且 a ≠ 0，那么 a 和 b 的关系是______。

答：a 和 b 互为相反数

练习3：计算：(-18) ÷ (+2) × (-5) ÷ (-3) = ______

答：-15（统一化乘法，负因数有 3 个，结果为负）

练习4：下列说法正确的是（ ）

A. 0 的倒数是 0
B. 负数的倒数还是负数
C. 倒数等于本身的数是 1
D. 除以一个数等于乘以这个数

练习5：计算：(-12) ÷ (+3) = ______

答：-4（异号相除得负）

1.10 有理数的乘方

📖 基础概念

乘方：n 个相同因数乘积的运算叫做乘方，记作 aⁿ。

符号规律：

  • 正数的任何次幂都是正数
  • 负数的偶次幂是正数
  • 负数的奇次幂是负数

📝 基础例题

计算：2⁴, (-2)⁴, -2⁴, (-2)³

2⁴ = 16

(-2)⁴ = 16（负数的偶次幂为正）

-2⁴ = -16（注意：这里指数只作用在 2 上）

(-2)³ = -8（负数的奇次幂为负）

⚠️ 易错提醒

  • "(-a)ⁿ" 和 "-aⁿ" 完全不同！前者底数包含负号，后者底数只是 a。
  • "-1" 的幂有规律：(-1)^偶数 = 1，(-1)^奇数 = -1。

📝 即时练习（5道题）

练习1：计算：(-1)^2024 + (-1)^2025 = ______

答：0（(-1)^偶数 = 1，(-1)^奇数 = -1，所以 1+(-1) = 0）

练习2：计算：-(-2)³ = ______

答：8（计算：-(-2)³ = -(-8) = 8）

练习3：计算：(-3)² - (-2)³ = ______

答：17（计算：9 - (-8) = 9+8 = 17）

练习4：下列说法正确的是（ ）

A. -a² 一定是负数
B. (-a)² 一定是正数
C. 如果 a² = 4，那么 a = 2
D. 如果 a² = 4，那么 a = ±2

练习5：计算：-2² × (-3)² = ______

答：-36（计算：-(2²) × (+9) = -4 × 9 = -36）

1.11 有理数的混合运算

📖 基础概念

运算顺序（非常重要！）：

  • 先乘方，再乘除，最后加减
  • 同级运算，从左到右依次进行
  • 有括号的，先算括号里面的

⚠️ 易错提醒

  • 一定要严格按运算顺序！
  • "-aⁿ" 和 "(-a)ⁿ" 在混合运算中最容易出错，一定要先判断底数。
  • 同级运算必须从左到右依次算。

📝 即时练习（5道题）

练习1：计算：(-2)³ + (-3) × [(-4)² + 2] - (-3)² ÷ (-2)

答：-57.5

练习2：计算：-1^100 - (0.5 - 2/3) ÷ 1/3 × [-2 - (-3)³]

答：11.5

练习3：计算：-2² ÷ 4/9 × (-2/3)² = ______

答：-8

练习4：计算：|-3| + (-2)³ - (-4) × 1/2 = ______

答：-3

练习5：计算：(-1)²⁰²⁴ × (-2)³ + (-3)² ÷ (-1/2) = ______

答：-40（计算：1×(-8) + 9×(-2) = -8-18 = -40）
